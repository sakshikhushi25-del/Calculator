# Calculator App 🧮

A simple and responsive calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

* Perform basic calculations (+, −, ×, ÷)
* Clear all input using **AC** button
* Delete last digit using **DEL**
* Supports decimal numbers
* Responsive and modern UI design
* Real-time display updates

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

---

## 📂 Project Structure

* `cal.html` → Structure of the calculator
* `cal.css` → Styling and layout
* `cal.js` → Functionality and logic

---

## ▶️ How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Open `cal.html` in your browser
4. Start using the calculator

## 🎯 How It Works

* Click numbers to input values
* Click operators (+, −, ×, ÷) to perform operations
* Press **=** to get the result
* Press **AC** to clear everything
* Press **DEL** to remove the last character

---

## ⚙️ Core Logic

* Uses JavaScript event listeners on buttons
* Stores user input in a string
* Uses `eval()` to calculate result
* Updates display dynamically using DOM
